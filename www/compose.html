Compose new music using the ark!

In this web app you have three choices:

  • Choose a prepared text and style

    • Choose from a set of input texts with pre-selected options for tone and meter.
    • You can still choose florid or simple style.
    • Several of the prepared texts include multiple sections with changes of tone and meter.

  • Choose a prepared text and make more musical choices

    • Choose from prepared input texts and apply your own choices of style, tone, and meter.

  • Write your own input text

    • Type in your own input text and set it in simple style.
