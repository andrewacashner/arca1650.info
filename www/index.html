ARCA MUSARITHMICA

a device for automatic music composition from 1650

This site presents Andrew Cashner’s digital implementation of a device for automatic music composition invented by Athanasius Kircher in 1650, along with research about the device and its history. Based on the specifications in Kircher’s Musurgia universalis (Rome, 1650), Book VIII, the core program was written in Haskell (2021), with a simple web interface in Javascript and PHP.

  • Generate your own new music using your own text or one of several preset texts, and hear it played back
  • Learn more about the Arca and Kircher’s concepts of computation and music
  • Learn about the people who used the Arca in the seventeenth and eighteenth centuries and what it meant to them, and see the evidence for the first known copy of the Arca in the New World.
  • Read the Haskell code and documentation to see how the Arca can be implemented in pure-functional programming.

This project was created on the ancestral land of the Onöndowa’ga:’ (Seneca) Nation, one of the Six Nations of the Haudenosaunee (Iroquois) Confederacy. Funding was provided by the University of Rochester.
